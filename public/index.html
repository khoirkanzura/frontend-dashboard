BankKu Dashboard

Proyek dashboard finansial ini sudah disiapkan untuk preview publik gratis. Versi penuh dengan Laravel backend dan routing akan berjalan optimal di hosting PHP, sedangkan halaman ini siap diakses melalui GitHub Pages.

Yang sudah siap

  • Halaman preview publik
  • Deploy otomatis dari GitHub
  • Gratis dan sederhana

Catatan

  • Fitur Laravel penuh membutuhkan hosting PHP
  • Halaman ini cocok untuk preview cepat
  • Deploy berikutnya bisa ditingkatkan nanti
